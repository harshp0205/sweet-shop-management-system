# Sweet Shop Management System

A full-stack TDD-based Sweet Shop Management System with user authentication, sweet inventory management, and purchase functionality.

## Project Overview

This project is built following Test-Driven Development (TDD) principles, implementing a complete sweet shop management system with:
- User authentication (registration & login)
- Sweet inventory management
- Purchase and restocking functionality
- Role-based access control (Admin features)

## Tech Stack

### Backend
- Node.js with TypeScript
- Express.js
- PostgreSQL Database
- JWT Authentication

### Frontend
- React with TypeScript
- Modern UI/UX Design
